# Money Divisor

Money Divisor is a lightweight web application that helps groups split shared expenses fairly. Simply add participants, record expenses, choose who paid and who participated, and the app calculates both the complete debt matrix and the minimum number of transfers required to settle everything.

## Features

- 👥 Add and remove participants.
- 💸 Record shared expenses.
- ✅ Select who paid each expense.
- 🎯 Choose exactly who participates in each expense.
- 📊 View the complete debt matrix.
- 🔄 Calculate the minimum number of payments needed to settle debts.
- 📁 Import and export data as JSON.

## How It Works

1. Add all participants.
2. Create expenses by entering:
   - Description
   - Amount
   - Who paid
   - Who shares the expense
3. The application automatically:
   - Calculates each participant's balance.
   - Generates the raw debt matrix.
   - Optimizes the settlements to minimize the number of transfers.
